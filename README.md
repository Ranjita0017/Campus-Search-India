# Campus Search-India
A simple web application that fetches and displays college information from the Hipolabs Universities API based on the Indian state entered by the user.

## Features
- Fetches university data from the Hipolabs Universities API.
- Search colleges by state name.
- Dynamically displays matching college names.
- Uses asynchronous API requests with Axios.
- Demonstrates DOM manipulation and event handling in JavaScript.
- Supports case-insensitive state search.

## Technologies Used
- HTML
- CSS
- JavaScript
- Axios
- Hipolabs Universities API

## Learning Outcomes
- Working with external APIs.
- Handling asynchronous operations using async/await.
- DOM manipulation with JavaScript.
- Event-driven programming.
- Filtering and displaying API data dynamically.
