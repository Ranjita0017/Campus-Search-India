# Campus Search-India
A simple web application that fetches and displays college information from the Hipolabs Universities API based on the Indian state entered by the user.

## Features
- Fetches college data using the Hipolabs Universities API.
- Search colleges by entering an Indian state name.
- Displays college names dynamically using JavaScript.
- Demonstrates API fetching and DOM manipulation.

## Technologies Used
- HTML
- CSS
- JavaScript
- Hipolabs Universities API
